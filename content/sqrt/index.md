# sqrt
C/C++에서 제곱근을 구해주는 함수입니다.

## 함수 원형과 사용법

### 헤더파일

C: <math.h>

C++: <cmath>

### 함수 원형

- double sqrt(double x)
- float sqrt(float x)
- long double sqrt(long double x)

### 사용법

매개변수로 들어온 숫자에 루트를 적용한 값을 반환합니다.

ex) 루트 9를 구하고 싶으면

```cpp
double ret = sqrt(9);
```
